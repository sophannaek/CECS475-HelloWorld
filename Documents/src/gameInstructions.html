Game Controls

Carrying out commands


How to Move

In order to move a troop/fleet, select the troop/fleet that you would like to move. Then click on the move button and then on the area in which you want to move to.


How to Attack

In order to attack a troop/fleet, select the troop/fleet that you would like to use to attack with and then click on the attack button. Then click on the province you would like to attack.


How to Hold

In order to Hold a troop/fleet, select the troop/fleet that you would like to use to Hold, then click on the hold button. If no order is made for a fleet/troop it will automatically be held for that round.
